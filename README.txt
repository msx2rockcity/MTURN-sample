---------------------------------------------

     MSX2 3D Wireframe Sample Program

     MTURN  Source file

            Created in 1993

---------------------------------------------

ROCKCITYを作る元になった、3D回転サンプルプログラムです
カーソルキーでMの文字が回転します

作成したのは高校3年の頃、3Dを表現するための練習プログラムだったのだと思います
これを作成したことを期に、ROCKCITY製作が始まったのだと記憶しております

簡単なサンプルプログラムですので、DeepWikiで簡単に仕組みが解るようにするために
github化しました

アセンブリ言語ファイルをアセンブルしてPROGRAM.BINというファイルで保存
ベーシックからbload "PROGRAM.BIN",rで実行します
